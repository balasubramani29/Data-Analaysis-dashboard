## Adidas Sales Analysis Dashboard (Power BI)
Project Overview

This project focuses on analyzing Adidas sales data using Power BI to understand business performance across different regions, product categories, retailers, and sales methods. The dashboard provides interactive visualizations that help identify key trends, sales distribution, and profitability.

The goal of this project is to transform raw sales data into meaningful insights that can support data-driven business decisions.

# Objectives

Analyze overall sales performance.

Identify top-performing regions and retailers.

Understand product category contribution to total sales.

Analyze sales distribution across different sales channels.

Monitor key metrics such as total sales, operating profit, and units sold.

Tools & Technologies Used

Power BI

Power Query

DAX (Data Analysis Expressions)

Data Visualization

Data Analysis Dashboard

The dashboard provides a comprehensive view of Adidas sales performance using interactive visuals and slicers.

Key KPIs

Total Sales: 120M

Operating Profit: 47.22M

Units Sold: 2M

Average Price per Unit: 45.22

Visualizations Used

Line Chart – Sales trend by year and quarter

Bar Chart – Sales by retailer

Donut Chart – Sales by sales method

Pie Chart – Sales distribution by product category

Bar Chart – Sales by product category

Clustered Column Chart – Sales by region and gender

Filters such as Region, City, and Invoice Date allow users to interactively explore the data.

Process

Data Collection
Obtained the Adidas sales dataset containing information about sales transactions, products, retailers, and regions.

Data Cleaning
Used Power Query to remove duplicates, handle missing values, and prepare the dataset for analysis.

Data Modeling
Structured the dataset and established relationships between tables to create an efficient data model.

Data Analysis using DAX
Created measures for:

Total Sales

Operating Profit

Units Sold

Average Price per Unit

Dashboard Development
Designed an interactive dashboard using different charts, cards, and slicers to make insights easy to understand.

Key Insights

The West region generates the highest sales compared to other regions.

Street Footwear is the top-selling product category.

West Gear retailer contributes the highest sales among retailers.

Online sales channel contributes a large share of total sales.

Sales increased significantly in later quarters of the year.
